# Claude Artifact Prompt — Perceptivity Homepage

You are building a premium cinematic website for **Perceptivity**, an AI perception OS for AEO/GEO.

Use the attached design pack assets and CSS.

## Brand direction

The visual style is:
- Interstellar atmosphere
- Nothing / Polestar minimalism
- Deep space black
- Warm film amber
- Dust beige
- Cold teal highlights
- Mist, plume, singularity, eye, citation graph

## Page goal

Make the visitor understand that the next brand battlefield is not SEO ranking — it is AI belief.

## Website sections

1. Hero  
   Full-bleed `mist-peacock-hero.svg` background.  
   Headline: “See how AI sees your brand.”  
   Subcopy: “Perceptivity tracks how answer engines form belief, cite sources, hallucinate claims, and recommend competitors.”

2. The shift  
   Big editorial typography.  
   Copy: “Search was a page. AI is a belief system.”

3. System loop  
   Sense → Simulate → Strike → Learn.  
   Use `agentic-loop-rings.svg` or `perception-plume-animated.svg`.

4. Answer graph  
   Use `answer-graph-field.svg`.  
   Explain citations, claims, rank, sentiment, hallucination, and answer drift.

5. Source trust  
   Use `source-constellation-map.svg`.  
   Explain which sources AI trusts.

6. India wedge  
   Use `vernacular-wave-field.svg`.  
   Explain Hinglish, Hindi, Tamil, Telugu, Marathi, Bengali, voice and local AI answers.

7. Execution OS  
   Glass cards: diagnose, assign, publish, measure.

8. Close  
   CTA: “Don’t optimize for blue links. Optimize for belief.”

## Implementation rules

- Use semantic HTML.
- Use `Inter Tight`, `Inter`, and `JetBrains Mono`.
- Use the CSS variables from `design.md`.
- Add slow scroll reveal animation.
- Avoid generic SaaS dashboards.
- Avoid stock imagery.
- Avoid neon cyberpunk.
- Make it feel expensive, quiet, sharp, and intelligent.
